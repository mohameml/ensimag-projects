computes the edit distance between A[0 .. lengthA-1] and B[0 .. lengthB-1]

Parameters
A: array of char represneting a genetic sequence A
lengthA: number of elements in A
B: array of char represneting a genetic sequence B
lengthB: number of elements in B
Returns
: edit distance between A and B }

editDistance_RecMemo is a memoized recursive immplementatioin of Needleman-Wunsch algorithm. It allocates the data structure for memoization table and calls the internal recursive function _editDistance_memo that fills in the memoization table.

If lengthA < lengthB, the sequences A and B are swapped.
