edit\+Distance\+\_\+\+Rec\+Memo is a memoized recursive immplementatioin of Needleman-\/\+Wunsch algorithm. It allocates the data structure for memoization table and calls the internal recursive function \+\_\+edit\+Distance\+\_\+memo that fills in the memoization table.

If lengthA $<$ lengthB, the sequences A and B are swapped.
